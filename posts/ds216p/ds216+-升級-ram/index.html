備戰DSM 7.0 , 為 DS216+ 升級到 8GB Ram

2021-09-19 1 分鐘閱讀 SynologyNASDIY

前言

DS216+ 只有 1 GB ram, 而且長期在跑 Plex media server, VPN Server, qBittorrent 等… 輕易就可佔滿記憶體, 拖慢整台機器, 所以決定在升級DSM 7.0 前把 RAM 升級 !

  • 注意事頂
    拆機升級ram , 會令 DS216+ 保養失效
    DS216+ 支授的記憶體規格為 **單條 DDR3L SODIMM**

Synolody 自家的 ram 應該有最好的相客性, 不過價錢比較貴, 所以我隨便選了g.skill 的 ram 。該 RAM 不在官方 QVL 中, 要實際安裝測試才知相容性如何。


拆機

首自把硬碟移除



轉到背面, 把5粒螺絲擰開



移除風扇罩



撬開2個卡榫



推出另外半邊



移除外罩



移除螺絲 把黑色支架移除



移除螺絲 把灰色支架移除


撕開金屬膠紙


移除4粒螺絲


就可把主版拉起, 並轉到背面


移除舊ram & 安裝新 ram

向外撥開卡榫



ram 會向上彈起, 將 ram 拿掉


把新 ram 斜插進去



向下按 會聽到「卡」一聲



把NAS還原裝回去



開機成功


測試

由於我用的 ram 不在 QVL 中, 所以用了 Synology Assistant 來測試 ram 的穩定性 , 過程中會自動重啟



可看到測試進程



測試完成後沒有發現問題



順手升級成 DSM 7
再去到「資源監控」> 「記憶體」查看 ram 使用率
剛登入沒有運行 media server 和下載器的 又狀況已佔用了1.4 GB RAM


拆下來的原裝 ram , 希望之後不會再派上用場!
